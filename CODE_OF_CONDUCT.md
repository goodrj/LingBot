# Code Of Conduct

LingBot is a small project, but the standard is still simple: be clear, kind, and useful.

## Expected Behavior

- Explain problems without attacking people.
- Keep feedback specific.
- Respect private account and task information.
- Help beginners feel welcome.

## Unacceptable Behavior

- Harassment or insults.
- Sharing private credentials, cookies, or task data.
- Pressuring someone to run automation on accounts they are not allowed to use.

## Reporting

If something in the project feels unsafe or inappropriate, open an issue or contact the maintainer privately.
